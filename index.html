Nature Forge
Curved Sounds Fieldscape Instrument

• Randomize Landscape creates a new natural soundscape.

• Journey mode automatically evolves the environment over time.

• Use Landscape Presets for instant atmospheres.

• Adjust individual track volumes to balance the mix.

• Master Volume controls the overall output level.

• Master Lowpass softens the sound for relaxing listening.

• Save Set stores the current mix in your browser.

• Load From Browser restores a saved mix.

• Export JSON saves a preset file.

• Import JSON loads a previously exported preset.

• Stop All immediately stops every playing sound.
